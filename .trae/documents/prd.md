## 1. 产品概述

面向独立恐怖游戏编剧的 Web 谜题草稿台，专注于将"阴间电台"段落转化为可落地的关卡需求。通过 AI 辅助生成电台谜面、线索链检查和结局提示预览，帮助编剧快速设计公平且有氛围感的解谜关卡。

- 核心价值：解决恐怖游戏谜题设计中"只剩氛围没有解法"的常见问题，提供从电台文本生成到线索验证的完整工具链。
- 目标用户：独立恐怖游戏编剧、关卡设计师。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|-----------|----------|
| 编剧用户 | 无需注册，本地使用 | 完整使用所有生成、检查、预览功能 |

### 2.2 功能模块

1. **电台谜面生成页**：参数配置区、关键词输入、生成结果展示
2. **线索链检查页**：线索列表、答案映射、完整性验证
3. **结局提示预览页**：玩家视角模拟、多场景反馈

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|----------|
| 电台谜面生成页 | 参数配置区 | 选择章节位置、玩家已知信息、广播口吻、恐怖强度 |
| 电台谜面生成页 | 关键词输入 | 输入3-5个恐怖元素关键词（如失踪主播、午夜频率、倒放童谣） |
| 电台谜面生成页 | 生成结果区 | 展示可播出的电台文本、解谜目标、玩家操作步骤 |
| 线索链检查页 | 线索列表 | 展示从电台文本中提取的所有暗示 |
| 线索链检查页 | 答案映射 | 将每条线索关联到具体答案（频道数字、收音机旋钮、磁带顺序等） |
| 线索链检查页 | 完整性验证 | 检查线索链是否完整、是否存在逻辑断层 |
| 结局提示预览页 | 玩家视角模拟 | 按步骤展开提示 |
| 结局提示预览页 | 多场景反馈 | 显示第一次收听、重复收听、触发失败后的不同反馈 |

## 3. 核心流程

用户在生成页配置参数并输入关键词，系统生成电台文本和解谜目标；随后进入检查页验证每条线索是否指向具体答案；最后在预览页模拟玩家体验，判断谜题公平性。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深夜黑 (#0a0a0f)、阴森绿 (#1a3a2a)、诡异红 (#8b0000)、静电灰 (#2a2a3a)
- **辅色调**：荧光绿 (#39ff14)、电台橙 (#ff6b35)
- **按钮风格**：做旧磨损效果，hover 时伴有轻微静电噪点动画
- **字体**：标题使用 'Creepster' 或 'Nosifer' 风格恐怖字体，正文使用 'VT323' 等复古终端字体，营造老旧电台+CRT显示器质感
- **布局风格**：暗色模式为主，卡片采用仿老式收音机/磁带盒的立体边框设计，页面带有轻微扫描线和噪点纹理
- **图标风格**：复古像素风图标，包含收音机、磁带、旋钮、天线、波形等元素

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 电台谜面生成页 | 参数配置区 | 下拉选择器带有老式收音机旋钮动画，滑动条模拟音量旋钮效果 |
| 电台谜面生成页 | 关键词输入 | 标签式输入，每个关键词显示为磁带标签样式 |
| 电台谜面生成页 | 生成结果区 | 模拟电台频谱可视化动画，文本逐字打字机效果 |
| 线索链检查页 | 线索列表 | 线索卡片带有连接线条，形成可视化线索链 |
| 线索链检查页 | 答案映射 | 拖拽式关联，线索与答案之间绘制连接线 |
| 线索链检查页 | 完整性验证 | 红绿灯状态指示器，断层处闪烁警告 |
| 结局提示预览页 | 玩家视角模拟 | 收音机界面交互，旋钮可旋转，带有"沙沙"静电效果 |
| 结局提示预览页 | 多场景反馈 | 不同状态下的收音机屏幕显示不同颜色的干扰波纹 |

### 4.3 响应式

桌面端优先设计，移动端自适应布局。触控设备优化旋钮和滑块的交互区域。

### 4.4 视觉特效

- 背景：深色背景带有 CRT 扫描线动画和轻微噪点纹理
- 转场：页面切换时的电视关机/开机效果
- 交互：按钮 hover 时的静电干扰效果，旋钮旋转时的调频音效视觉化
- 状态：错误/警告时的屏幕闪烁和颜色失真
